This page is a hub for our **step-by-step guides**. Each guide focuses on decision-making and process—not predictions.

## How to use these guides

The goal is not to read everything. The goal is to build a simple base you can actually follow.

1. Start with your intent (long-term investing, active trading, crypto).
2. Keep size small and avoid leverage when you’re learning.
3. Pick one setup (platform + routine) and stick to it for 90 days.

## Recommended path (beginners)

If you’re starting from scratch, this path is usually the fastest way to build clarity:

1. [How to Start Investing](/guides/start-investing/) — goals, horizon, and risk basics.
