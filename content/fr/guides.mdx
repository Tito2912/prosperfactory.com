Cette page regroupe nos **guides pas-à-pas**. L’approche est volontairement orientée “process” (décision, risques, checklist) plutôt que prédiction.

## Comment utiliser ces guides

L’objectif n’est pas de lire “tout”. L’objectif est de construire un socle simple que vous pouvez tenir.

1. Commencez par l’intention (investir long terme, trader actif, crypto).
2. Appliquez une règle : **petites tailles** + **pas de levier** si vous débutez.
3. Choisissez un seul setup (plateforme + routine) et tenez-le 90 jours.

## Parcours recommandé (débutant)

Si vous partez de zéro, voici un parcours efficace et progressif :

1. [Débuter l’investissement : méthode simple](/fr/guides/debuter-investissement/) — objectifs, horizon, bases du risque.
2. [Choisir une plateforme : checklist](/fr/guides/comment-choisir-une-plateforme/) — produit (actif vs CFD), frais, retraits, sécurité.
3. [Lire un graphique](/fr/guides/comment-lire-un-graphique/) — unités de temps, tendance, zones, routine.
4. [Outils d’analyse technique](/fr/guides/meilleurs-outils-analyse-technique/) — quoi utiliser (et quoi éviter) selon votre workflow.
5. Option crypto : [DCA crypto](/fr/guides/dca-crypto/) — règles, frais, sécurité et discipline.
